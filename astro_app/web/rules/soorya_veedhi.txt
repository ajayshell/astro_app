Rule for Soorya Veedhi:
Rule 1: Rishaba, Mituna, Kadaga, Simha <-- these four raasi are called mesha veedhi
Rule 2: Kanni, Thula, Meena, Mesha <-- these four are called rishaba raasi
Rule 3: Virchigam, Dhanush, Magara, Kumba <-- these are called mithuna veedhi

- Find the Raasi in which Surya is placed in D1
- If it is in Rule 1, then count from Aarudum clockwise to Mesha
- If it is in Rule 2, then count from Aarudum clockwise to Rishaba
- If it is in Rule 3, then count from Aarudum clockwise to mituna
The result should be called Soorya Veedhi Count (SVC)


Next, count from Udayam, clockwise for SVC number of squares and then write 'Kavippu' in that square.


Rule for calculating the degrees for Aarudom in the chart:
Pick up the minutes of the time of birth and multiply it by 6 = degree of the Aarudom. Call this Ad
From Mesham, subtract 30 degrees from Ad for each square traversed, including Mesham. When writing Aarudom write the reminder next to it.

Rule for calculating degrees for Kaippu.
degrees for Kavippu = 30 - degrees of Aarudom.
Calculate this and print it next to Kavippu (wherever Kavippu is present)

- Rule for placement of Jamagraha
    - Given time in 12 hour format - 6hours -> A
    - For A -> convert to minutes. e.g: 3.12 should be 3*60 + 12. Call this B
    - C = B/2
    - 360-C. Call it D.
    - From Aries (inclusive), clockwise, count forward and place the value of D in the outer side of the 
    square and write the ‘day of the god’ in the square. -> Call this square as sqE.
    - Fill all the the outer blue boxes with other gods. Start from Saturn, anticlockwise, fill each of the blue
    box as a series as follows:
        - Sun, Mars, Jupiter, Mercury, Venus, Saturn, Moon, Snake.
    example: time is 7.12pm. Day = Saturday
        A = 7 - 6 = 1
        B = 1*60+12 = 72
        C = B/2 = 36
        D = 360 - C = 324
        Now, starting from Aries, inclusive, move 30degrees for each square. So 10 squares and remainder of 8 degrees
        Write the 'god of day' and 324deg in that square.
