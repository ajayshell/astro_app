# References

Drop zone for source material used to define/verify the app's astrological
rules: photos or scans of book pages, reference chart screenshots, etc.

This is separate from `web/rules/`, which holds material that's already
tied to a specific implemented rule and cited by exact filename in code
comments and `PROJECT_BRIEF.md` (e.g. `web/rules/soorya_veedhi.txt`). Once
something dropped here becomes the basis for an implemented feature, it's
worth moving/copying it into `web/rules/` and citing it the same way.

Not linked from the running app, and not itself a source of truth for any
UI text — just supporting material.
